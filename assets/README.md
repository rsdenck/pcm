# PCM Assets

This directory contains project assets including logos and images.

## Logo Files

- `pcm-logo.png` - Main PCM logo (800x200px recommended)
- `pcm-logo-icon.svg` - Icon version of the logo

## Usage

The main logo is displayed at the top of the README.md file:

```markdown
<img src="https://raw.githubusercontent.com/rsdenck/pcm/main/assets/pcm-logo.png" alt="PCM Logo" width="800"/>
```

## Creating the Logo

To create the logo image from the provided design:

1. Use the Proxmox-style logo with orange (#ff7a00) and white colors
2. Black background (#000000)
3. Text: "PROXMOX CENTER MANAGER"
4. Export as PNG at 800x200px minimum
5. Save as `pcm-logo.png` in this directory
